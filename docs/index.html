ITエンジニアのLT大会

ライトニングトークとは、カンファレンスやフォーラムなどで行われる短いプレゼンテーションのことです。

このような方にオススメです！

  • ライトニングトークの経験を積む機会を増やしたい
  • 役に立つ情報を共有したい
  • 刺激をもらってモチベーションを高めたい

発表者は5分程度で、好きなことをプレンゼンします。
LT後、参加者で軽く質疑応答やディスカッション (10分くらい) して、次の発表者にバトンタッチ…を繰り返します。

全部で、4本前後のLTを予定しています。

#LT大会 のツイート

登壇者を募集しています！

テーマは自由です。

  • 得意分野の最新情報
  • 個人で開発しているツール・サービスの紹介
  • おすすめライフハック
  • 自己PR... などなど

複数テーマでも構いませんし、とりあえずテーマ未定でもOKです！

登壇希望の方は参加登録時にコメントいただくか、ハッシュタグ #LT大会でツイートしてください。
